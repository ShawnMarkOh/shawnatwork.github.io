Tool Selection
This page.
Airport Tool
Search airports via IATA codes (E.g DAY, PHL, CLT, DCA, Etc.) or their city names. Useful for knowing the time zone or weather at an airport.
Phonetic Tool
Type letters to have them converted to the NATO Phonetic Alphabet in real time. Useful for record locators.
Error Codes Tool
Save "smm_ercd.lis" from the error codes report anywhere you'd like (usually your desktop) and select it with the program to analyze the error code file and ensure all error codes are on their correct settings.
Deadheads Tool
Save "smm_dead.lis" from the deadhead report. The report can be ran for any length of time but is only useful for as long as crewmember schedules have been built for. Output will be any flights built into schedules that have an extra space before a three digit flight number.
Time Tool
Four clocks that display Eastern, Central, Mountain, and Zulu time.
Swap Finder Tool
Run and save a CrewTrac Moveup report and input the "In" and "Out" times for the crewmember you're trying to swap.
